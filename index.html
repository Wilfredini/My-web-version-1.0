O mně

Mým snem je stát se Front End developerem.

Absolvoval jsem kurz Webrebel, kde jsem se naučil základy HTML, CSS, základy responzivního web designu a Java Scriptu. Jsem career switcher. Nestudoval jsem školu s IT zaměřením. Pro tvorbu webů a webových aplikací jsem se nadchl a tvrdě pracuju na tom , aby se můj sen stal skutečností.Již několik měsíců se intenzivně věnuju studiu o moderním web designu a tvorbě webů.Toto je můj první projekt mimo kurz Webrebel.

Informace čerpám z kurzu Webrebel a youtube tutoriálů... Weby tvořím v programu Visual code studio a pomáhám si úpravou fotek ve Fotoshopu. Doufám, že vás má práce zaujme.

Moje služby

Jsem amatérský Front End developer. Tvorbou webů se neživím, ale je to můj koníček, kterému se věnuji ve volném čase...Vytvořím vám jednoduché stránky s konkatkním formulářem na míru. Mé stránky jsou plně responzivní a jsou uživatelsky přívětivé na všech zařízení. Pokud potřebujete upravit váš stávající web, dát mu nový kabátek a aby se zobrazoval hezky na všech zařízeních,tak jsem to právě já kdo by vám mohl pomoci. Vašim projektům se věnuji na plno a každý jednotlivý projekt je pro mě nesmírně důležitý především aby jste byli s mou prací spokojení a já jsem měl v portfoliu od vás působivé reference. Pomocí HTML,CSS a Java Scriptu vám vytvořím interaktivní stránky s designem podle vaších představ. Pomozte mi stát se profesionálem a pokud vás zaujali mé stránky, svěřte mi svůj projekt který společně vytvoříme tak, že budete jistě spokojeni.

Web design

Řeknete mi vaše představy o vizuální stránce vašeho webu a vytvořím vám web, který bude vypadat hezky na PC, tabletu i mobilním zařízení.

Galerie fotografií

Pomocí Java Scriptu budou mít vaše stránky a obrázkové galerie dynamický a moderní vzhled.

Stylované formuláře

Jsem kreativní a takový obyčejný kontaktní formulář v moderním stylu dokáže stránkám dodat moderní vzhled.

Oprava - nastavení stávajícího webu

Máte stránky už hotové a chcete jenom upravit některé položky, popřípadě změnit vizuál? Ozvěte se a spolu se na to podíváme.

Vašemu projektu věnuju veškerý čas

Jelikož jsou vaše projekty zároveň moje reference pro budoucího zaměstnavatele, můžete se spolehnout, že svou práci odvedu na 100% a budu se snažit dosáhnout vaší maximální spokojenosti.

Úspora financí

Jsem na začátku své kariéry Front End developera a moje ceny pro vás budou určitě zajímavé. Tento web slouží jako prezentace mých schopností.
